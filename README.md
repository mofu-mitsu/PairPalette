# Pair Palette Territory

キャラと並んで遊ぶ、2vs2の陣地取りパズル試作です。

## 遊び方

- `index.html` をブラウザで開く
- プレイヤー、サポート、敵A、敵Bを自由に選ぶ
- `開始` を押す
- 盤面クリック、WASD、矢印キーで移動
- Spaceまたはスキルボタンでスキル使用
- 75秒後、陣地率が高いチームの勝ち

## 現在の仕様

- 15 x 11 のパネル盤面
- プレイヤー + サポートキャラ vs 敵キャラA + 敵キャラB
- 歩いた場所と周辺パネルを自陣色にする
- サポートと敵キャラは自動で移動
- キャラごとの固有処理ではなく、役割ごとのスキルを使う
- 選択可能キャラは現在80人
- キャラ画像は `assets/characters/キャラID.png` を置くと自動表示

## 役割

- `attack`: 広範囲を一気に塗る
- `support`: 自分と相棒の周辺を回復する
- `guard`: 空白地帯を少しずつ固める
- `speed`: 遠い場所へ跳んで塗る
- `trick`: 敵陣を確率で反転する
- `technical`: 敵陣の薄い場所を解析して奪う

## キャラ追加方針

キャラを増やす時は、`script.js` の `characters` 配列へ追加します。

必要な項目:

- `id`: 半角英数字の一意ID
- `name`: 表示名
- `className`: 学年・組など
- `role`: 上記6種類のいずれか
- `type`: MBTI / ソシオニクスなど
- `line`: 相棒欄やログに出る短いセリフ
- `motif`: モチーフ

300人規模に増やす場合は、キャラデータだけ `characters.js` に分離すると保守しやすくなります。

## 画像追加

例: みりんてゃの画像を入れる場合

```text
assets/characters/mirin.png
```

画像がないキャラは、名前の頭文字を使った仮アバターで表示されます。
